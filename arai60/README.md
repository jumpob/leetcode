# arai60

[ソフトウェアエンジニアリング協会](https://www.swe.or.jp/)のコーディング練習会の題材である[arai60](https://1kohei1.com/leetcode/)の回答を残す

## 使い方

* 問題ごとにブランチを切る。
  * レビュー時はプルリクURLを共有するので。
* ステップ別(※後述)に回答ファイルを作成する。

## 進め方

ソフトウェアエンジニアリング協会で提示されるルールに則り、以下3ステップで進める。

### STEP1

* 答えを見ずに考える。5分悩んで出なかったら答えを見る。
* 答えを見て理解したら再度書く。また悩んで答えを見てしまったら1からやり直し。
* 悩まず書けるようになったらクリア。

### STEP2

* 見やすいコードにできるだけ整える。
* 動こコードができればクリア。

### STEP3

* 1から何も見ずに、時間を測りながら書く。アクセプトされたら消す。これを3回繰り返す。
* 10分以内に1回もエラーを出さずに書ける状態が3回続けばクリア。
